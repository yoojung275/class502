- 자바스크립트 함수

#기본형태
function 함수명 (매개변수, ..... ) {
	// 실행 코드 ...

	return 반환값;
}

!자바스크립트의 함수는 객체이다!

#용어정리
x : 매개변수
add : 변수 - 함수 객체의 주소값 
console.dir(...) : 객체의 속성과 값 형태 출력
[[Prototype]] : 프로토타입 체인 - 상속 관계 링크
						__proto__ 속성을 통해서 접근
EC (Execution Context) : 실행 가능 객체
                               자바스크립트의 함수는 객체라서 바로 실행이 안됌, 그래서 번역(평가) 과정에서 EC를 사용해야 실행이 됨.
                               ex) window (global EC)
유효범위 체인(Scope) : 전역 유효 범위와 지역 유효 범위로 나뉨
                             1) var : 함수 지역이 유효범위 
                                       var i = 8 이래 고정해놓고
                                       var i = 99 이래 한번 더 설정해도 오류가 안나고 99로 바뀌기 때문에 var는 잘 안씀
                             2) let : 변수( 값 변경이 가능)
                             3) const : 상수( 값 변경이 불가)
                             -> 변수는 기본적으로 const로 정의, 변경이 필요한 경우만 let으로 사용
일등 함수 : 로마에서 찐로마인과 식민지 일등시민을 동등하게 시민권줬듯이
               변수와 함수를 동등하게 취급, 함수 == 값
		1) 매개변수로 함수 객체를 사용 
		2) 반환값으로 함수 객체를 사용(클로저...)
